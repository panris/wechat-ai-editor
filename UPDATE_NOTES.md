# 🎉 重大更新：豆包客户端集成版

## 更新概览

我们完全重构了 WeChat AI-Editor，从 **API Key 方式**升级到**豆包客户端直接集成**，实现了：

- ✅ **零配置**：无需申请 API Key
- ✅ **完全免费**：无任何使用限制
- ✅ **体验更好**：在插件内直接与豆包对话
- ✅ **功能更强**：5 大专业 Prompt 模板

## 核心变更

### 1. 用户体验优化

#### 之前（API Key 版）
```
1. 访问火山引擎控制台
2. 注册账号并实名认证
3. 申请 API Key
4. 在插件中配置 API Key
5. 测试连接
6. 开始使用
```

#### 现在（集成版）
```
1. 安装插件
2. 注册豆包账号（支持微信登录）
3. 开始使用
```

### 2. 功能变化

| 功能 | API Key 版 | 集成版 |
|------|-----------|--------|
| **对话方式** | 文本输入框 | 嵌入豆包网页版 |
| **Prompt 输入** | 手动输入 | 5 大模板 + 自定义 |
| **内容传输** | 自动调用 API | 复制粘贴 |
| **历史记录** | 插件本地 | 豆包云端 |
| **功能限制** | 受 API 配额限制 | 无限制 |
| **成本** | 按量计费 | 完全免费 |

### 3. 技术架构变化

#### 之前
```
Chrome Plugin → Background Script → Doubao API → 返回结果
```

#### 现在
```
Chrome Plugin → 嵌入豆包 iframe → 用户直接交互
```

```
   ↓
6. 粘贴分析结果
   ↓

```

## 常见问题

### Q: API Key 版还能用吗？

**A**: 可以。但我们推荐使用集成版，体验更好且完全免费。

### Q: 如何切换回 API Key 版？

**A**: 在 Git 中切换到 `api-key-version` 分支（如果有的话），或查看之前的提交记录。

### Q: 豆包 iframe 加载失败怎么办？

**A**: 点击"在新窗口打开豆包"按钮，在独立窗口使用。

### Q: Prompt 模板可以自定义吗？

**A**: 可以！在 `content.js` 中修改 `PROMPT_TEMPLATES` 对象。

### Q: 后端服务还需要启动吗？

**A**: 不需要。日常使用插件即可，后端服务变为可选。
