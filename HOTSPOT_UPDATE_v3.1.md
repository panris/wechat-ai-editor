## 📝 使用说明

### 安装插件
1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `~/Desktop/WeChat-AI-Editor-With-Hotspot` 文件夹

### 使用热点功能

#### 方式1：查看热点
```
1. 打开微信公众号编辑页面
2. 点击右下角🤖按钮
3. 点击"🔥 热点"Tab
4. 选择热点源（微博/知乎/百度）
5. 浏览热点列表
```

#### 方式2：基于热点创作
```
1. 在热点列表中点击感兴趣的话题
2. 查看热点详情
3. 点击"✨ 基于此热点创作"按钮
4. 自动切换到创作Tab，Prompt已生成
5. 点击"🚀 打开豆包对话"
6. 在豆包中粘贴Prompt进行创作
7. 复制结果，点击"➜ 粘贴到编辑器"
```

#### 方式3：分析热点
```
1. 在热点详情页点击"🔍 分析热点"
2. 生成深度分析Prompt
3. 使用豆包进行热点分析
4. 了解热点背景、争议点、创作角度等
```

## 🎯 生成的Prompt示例

### 创作Prompt
```
请基于以下热点话题创作一篇公众号文章：

热点标题：[热点标题]

要求：
1. 紧扣热点，找准切入角度
2. 标题要有吸引力，15-25字
3. 内容1500-2000字
4. 结构清晰，有观点有案例
5. 语言生动，易于传播
```

### 分析Prompt
```
请深度分析以下热点话题：

热点标题：[热点标题]

分析维度：
1. 热点背景和起因
2. 核心讨论点和争议
3. 受众情绪和立场
4. 传播特点和趋势
5. 适合的创作角度
6. 风险提示和注意事项
```
