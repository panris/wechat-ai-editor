## 🚀 快速开始

### 第一步：安装插件

```bash
# 克隆项目
git clone https://github.com/your-repo/wechat-ai-editor.git
cd wechat-ai-editor/chrome-extension

# 构建插件
node build.js
```

在 Chrome 中加载插件：
1. 打开 `chrome://extensions/`
2. 开启"开发者模式"（右上角开关）
3. 点击"加载已解压的扩展程序"
4. 选择 `chrome-extension/dist` 文件夹

### 第二步：注册豆包账号

访问 [豆包官网](https://www.doubao.com/chat/)，使用微信/手机号注册账号（完全免费）

### 第三步：开始使用

1. 打开[微信公众号后台](https://mp.weixin.qq.com/)
2. 进入图文素材编辑页面
3. 点击右下角 🤖 悬浮按钮
4. 在侧边栏与豆包对话或使用快速模板

## 📖 使用场景

### 场景 1：分析竞品文章 📊

```
1. 复制竞品文章链接
2. 点击"创作工具" → "文章解构分析"
3. 在豆包中粘贴链接
4. 获取 7 维度深度分析
```

**分析维度**：核心主题、目标受众、痛点挖掘、文章结构、情绪基调、爆款元素、内容密度

### 场景 2：快速生成初稿 ✍️

```
1. 基于分析结果
2. 点击"生成原创初稿"模板
3. 在豆包中补充要求
4. 获取 1500-2000 字原创文章
5. 一键粘贴到编辑器
```

### 场景 3：优化标题 🎯

```
1. 点击"标题优化"模板
2. 输入文章主题
3. 获取 10 个爆款标题
4. 选择最合适的使用
```

### 场景 4：内容扩写 💡

```
1. 选中需要扩写的段落
2. 点击"扩写润色"模板
3. 粘贴原文到豆包
4. 获取扩写后内容（2-3倍）
```

### 场景 5：提取金句 🔍

```
1. 点击"提取金句"模板
2. 粘贴完整文章
3. 获取 10 条可传播金句
4. 用于海报、朋友圈宣传
```

## 💡 5 大 Prompt 模板

### 1. 📊 文章解构分析

深度分析文章的 7 个维度，提取核心要素和爆款基因。

**适用场景**：研究竞品、学习优秀案例、找寻创作灵感

### 2. ✍️ 生成原创初稿

基于分析结果，生成结构相似但内容原创的全新文章。

**适用场景**：快速起草、保持风格、提高效率

### 3. 🎯 标题优化

生成 10 个符合公众号爆款特征的标题，按点击率排序。

**适用场景**：标题不够吸引、想要A/B测试、寻找灵感

### 4. 💡 扩写润色

对指定段落进行扩写，增加案例、数据和生动描述。

**适用场景**：内容不够丰满、需要增加细节、提升可读性

### 5. 🔍 提取金句

从文章中提取 10 条具有传播价值的金句。

**适用场景**：制作海报、朋友圈宣传、公众号导语

## 🎓 最佳实践

### 1. 先分析，后生成

不要直接让 AI 写文章，先用"文章解构"分析竞品，再基于分析生成初稿。

### 2. 多轮对话优化

不满意第一次结果？继续追问："标题再短一些"、"语气更专业"、"增加一个案例"

### 3. 融入个人观点

在 Prompt 中加入你的独特见解和案例，让 AI 生成的内容更有个性。

### 4. 人工审核必不可少

AI 生成的内容仅供参考，必须经过人工审核和调整后再发布。

### 5. 遵守平台规范

生成的内容需符合微信公众号运营规范，不得发布违规内容。

## ❓ 常见问题

### Q: 完全免费吗？有使用限制吗？

**A**: 完全免费，无任何使用限制。豆包本身对个人用户免费开放。

### Q: 需要安装后端服务吗？

**A**: 不需要。日常使用插件即可，后端服务和爬虫是可选的独立工具。

### Q: 豆包窗口加载慢怎么办？

**A**: 点击"在新窗口打开豆包"按钮，在独立窗口使用完整功能。

### Q: 可以用于其他 AI 工具吗？

**A**: 可以！Prompt 模板是通用的，复制后可用于 ChatGPT、Claude、文心一言等。

### Q: 生成的内容会重复吗？

**A**: 不会。每次生成都是全新的内容，且豆包有防重复机制。

### Q: 如何保存对话历史？

**A**: 豆包会自动保存所有对话历史，可在豆包官网查看。
