# 🔥 热点功能说明

## 📋 功能概述

新增"热点"Tab，实时展示社会热点话题和相关文章，帮助创作者快速把握热点、提升内容时效性。

## ✨ 核心功能

### 1. 多源热点聚合
- **微博热搜** - 最新微博热搜话题
- **知乎热榜** - 知乎热门讨论
- **百度热榜** - 百度搜索热点

### 2. 实时更新
- 自动获取最新热点数据
- 显示热度指标
- 支持手动刷新

### 3. 热点详情
- 点击查看热点详情
- 显示热度、来源、描述等信息
- 提供相关操作按钮

### 4. 智能创作
- **基于热点创作** - 一键生成热点文章创作Prompt
- **热点分析** - 深度分析热点背景、争议点、创作角度

## 🔧 技术实现

### 数据源
使用免费的热点API服务：
- API地址：`https://api.vvhan.com/api/hotlist`
- 支持多个热点源
- 无需认证，免费使用

### 数据格式
```javascript
{
  "success": true,
  "data": [
    {
      "title": "热点标题",
      "hot": "1234567", // 热度值
      "url": "链接",
      "desc": "描述"
    }
  ]
}
```

### 关键函数

#### 1. `switchTab(tabName)`
切换创作/热点Tab

#### 2. `loadHotspots(source)`
加载指定源的热点数据

#### 3. `fetchHotspots(source)`
从API获取热点数据

#### 4. `renderHotspotList(hotspots, source)`
渲染热点列表

#### 5. `showHotspotDetail(hotspot, source)`
显示热点详情

#### 6. `createFromHotspot()`
基于热点生成创作Prompt

#### 7. `analyzeHotspot()`
生成热点分析Prompt

## 📝 使用流程

### 场景1：查看热点
1. 打开插件侧边栏
2. 点击"🔥 热点"Tab
3. 选择热点源（微博/知乎/百度）
4. 浏览热点列表

### 场景2：基于热点创作
1. 点击感兴趣的热点
2. 查看热点详情
3. 点击"✨ 基于此热点创作"
4. 自动切换到创作Tab
5. 生成的Prompt已准备好
6. 点击"打开豆包对话"进行创作

### 场景3：分析热点
1. 点击热点进入详情
2. 点击"🔍 分析热点"
3. 生成深度分析Prompt
4. 使用豆包进行分析

## 🎯 Prompt模板

### 创作Prompt
```
请基于以下热点话题创作一篇公众号文章：

热点标题：[热点标题]

要求：
1. 紧扣热点，找准切入角度
2. 标题要有吸引力，15-25字
3. 内容1500-2000字
4. 结构清晰，有观点有案例
5. 语言生动，易于传播
```

### 分析Prompt
```
请深度分析以下热点话题：

热点标题：[热点标题]

分析维度：
1. 热点背景和起因
2. 核心讨论点和争议
3. 受众情绪和立场
4. 传播特点和趋势
5. 适合的创作角度
6. 风险提示和注意事项
```

## 📊 数据说明

### 热度值格式化
```javascript
function formatHotValue(value) {
  if (value > 10000) {
    return (value / 10000).toFixed(1) + '万';
  }
  return value.toString();
}
```

### 时间显示
使用`toLocaleTimeString('zh-CN')`显示本地化时间

## ⚠️ 注意事项

### 1. API限制
- 免费API可能有请求频率限制
- 建议实现请求缓存和防抖
- 失败时显示友好错误提示

### 2. 数据实时性
- 热点数据非实时更新
- 建议每次切换Tab或手动刷新获取最新数据
- 显示更新时间供用户参考

### 3. 内容审核
- 热点话题可能涉及敏感内容
- 建议创作时注意内容合规性
- 提供风险提示

### 4. 性能优化
- 热点列表限制显示前50条
- 使用虚拟滚动优化长列表渲染
- 图片懒加载（如果有）
