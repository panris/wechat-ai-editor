# WeChat AI-Editor 使用指南（豆包集成版）

## 🎉 重要更新

**无需 API Key！** 本版本直接集成豆包客户端，完全免费使用，无需任何配置。

## ✨ 新特性

- ✅ **零配置启动**：无需 API Key，安装即用
- ✅ **豆包窗口集成**：插件内直接与豆包对话
- ✅ **5 大 Prompt 模板**：一键生成专业创作提示词
- ✅ **快速复制粘贴**：豆包 ↔ 编辑器无缝衔接
- ✅ **完全免费**：无任何使用限制

## 🚀 快速开始

### 第一步：安装插件

```bash
cd chrome-extension
node build.js
```

在 Chrome 中加载 `dist` 目录：
1. 打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `dist` 文件夹

### 第二步：注册豆包账号

访问 [豆包官网](https://www.doubao.com/chat/)，注册/登录账号（支持手机号、微信等方式）。

### 第三步：开始使用

1. 打开微信公众号编辑页面
2. 点击右下角 🤖 悬浮按钮
3. 在侧边栏与豆包对话或使用快速模板

## 📖 使用场景

### 场景 1：分析竞品文章

1. 点击侧边栏的"竞品追踪" Tab
2. 点击"添加文章链接"按钮
3. 输入竞品文章URL
4. 系统自动生成分析 Prompt 并复制到剪贴板
5. 切换到"豆包对话" Tab，粘贴并发送
6. 豆包返回深度分析结果

### 场景 2：快速生成初稿

1. 点击"创作工具" Tab
2. 选择"生成原创初稿"模板
3. Prompt 自动复制
4. 切换到"豆包对话" Tab，粘贴
5. 补充分析结果后发送
6. 点击"从豆包复制"→"粘贴到编辑器"

### 场景 3：标题优化

1. 点击"创作工具" Tab
2. 选择"标题优化"模板
3. 切换到豆包，粘贴你的文章主题
4. 豆包生成 10 个爆款标题供选择

### 场景 4：内容扩写

1. 在编辑器中选中需要扩写的段落
2. 复制到剪贴板
3. 使用"扩写润色"模板
4. 在豆包中粘贴原文，获取扩写结果

### 场景 5：提取金句

1. 使用"提取金句"模板
2. 粘贴完整文章内容
3. 豆包提取 10 条可传播金句
4. 用于海报、朋友圈等宣传

## 🛠️ 功能详解

### 豆包对话面板

- **直接对话**：在插件内嵌的豆包窗口中直接交流
- **新窗口打开**：点击按钮在独立窗口使用完整豆包功能
- **从豆包复制**：快速读取剪贴板内容
- **粘贴到编辑器**：一键插入微信编辑器

### 创作工具面板

提供 5 个专业 Prompt 模板：

#### 1. 📊 文章解构分析

深度分析文章的 7 个维度：
- 核心主题
- 目标受众
- 痛点挖掘
- 文章结构
- 情绪基调
- 爆款元素
- 内容密度

#### 2. ✍️ 生成原创初稿

基于分析结果，生成全新文章，要求：
- 结构相似，内容原创
- 融入最新热点
- 爆款标题特征
- 1500-2000 字
- 贴近目标受众

#### 3. 🎯 标题优化

生成 10 个吸引眼球的标题：
- 包含数字、疑问、对比
- 激发好奇心
- 避免标题党
- 按点击率排序

#### 4. 💡 扩写润色

对指定段落扩写：
- 增加具体案例
- 丰富论据数据
- 加入生动比喻
- 扩写 2-3 倍

#### 5. 🔍 提取金句

提取可传播金句：
- 10 条独立金句
- 有启发性、共鸣性
- 15-50 字
- 标注类型（励志/干货/情感）
- 按传播价值排序

### 自定义 Prompt

如果预设模板不满足需求，可以：
1. 在"自定义 Prompt"框中输入你的需求
2. 点击"发送到豆包"
3. 自动切换到豆包对话窗口

## 💡 使用技巧

### 技巧 1：多轮对话优化

不满意第一次生成结果？继续追问：
- "标题再短一些"
- "语气更专业一点"
- "增加一个案例"

### 技巧 2：组合使用

典型工作流：
1. 先用"文章解构"分析竞品
2. 再用"生成初稿"创作内容
3. 用"标题优化"润色标题
4. 用"提取金句"生成宣传文案

### 技巧 3：保存常用 Prompt

将你常用的 Prompt 保存到豆包的"我的对话"，下次直接调用。

### 技巧 4：批量处理

打开多个豆包对话窗口，并行处理多篇文章的分析和生成。

## ❓ 常见问题

### Q: 豆包窗口无法加载？

**A**: 检查网络连接，或点击"在新窗口打开豆包"独立使用。

### Q: 剪贴板功能不生效？

**A**: 确保授予了插件的剪贴板权限（安装时会自动请求）。

### Q: 生成的内容不够原创？

**A**: 在 Prompt 中强调"完全原创"，并提供你自己的案例和素材。

### Q: 如何保存豆包的对话历史？

**A**: 豆包会自动保存对话历史，可在豆包官网查看所有历史记录。

### Q: 可以用其他 AI 工具吗？

**A**: 可以！Prompt 模板是通用的，复制后可用于 ChatGPT、Claude、文心一言等。
