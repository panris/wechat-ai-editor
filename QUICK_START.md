## 🔧 常见问题快速解决

### 问题1：热点功能显示"加载失败"

**解决方案：**
```bash
# 确保后端服务正在运行
cd backend
mvn spring-boot:run

# 重新加载Chrome扩展
# 访问 chrome://extensions/ 并点击刷新按钮
```

### 问题2：配图预览界面不显示

**解决方案：**
```bash
# 1. 检查后端服务
curl http://localhost:8080/api/ai/health

# 2. 重新加载扩展
# 访问 chrome://extensions/ 并点击刷新按钮
```

### 问题3：后端端口被占用

**解决方案：**
```bash
# 杀掉占用8080端口的进程
lsof -ti:8080 | xargs kill -9

# 重新启动
mvn spring-boot:run
```
