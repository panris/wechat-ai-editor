## 🚀 快速开始

### 系统要求
- **操作系统**: macOS 10.14+ / Windows 10+ / Linux
- **浏览器**: Chrome 90+ / Edge 90+
- **Java**: JDK 17+
- **Maven**: 3.6+

### 第一步：启动后端服务

```bash
# 进入后端目录
cd backend

# 首次运行：下载依赖并编译
mvn clean install

# 启动服务
mvn spring-boot:run
```

等待看到服务启动成功提示：
```
================================================
🤖 WeChat AI-Editor Backend Started
================================================
```

### 第二步：安装Chrome扩展

1. 打开 `chrome://extensions/`
2. 开启"开发者模式"（右上角开关）
3. 点击"加载已解压的扩展程序"
4. 选择 `chrome-extension` 文件夹

### 第三步：开始使用

1. 打开[微信公众号后台](https://mp.weixin.qq.com/)
2. 进入图文素材编辑页面
3. 点击右下角 🤖 悬浮按钮
4. 开始创作！

**详细文档**: [完整安装指南](INSTALL_GUIDE.md) | [快速上手](QUICK_START.md)

## 💡 5 大 Prompt 模板

### 1. 📊 文章解构分析

深度分析文章的 7 个维度，提取核心要素和爆款基因。

**适用场景**：研究竞品、学习优秀案例、找寻创作灵感

### 2. ✍️ 生成原创初稿

基于分析结果，生成结构相似但内容原创的全新文章。

**适用场景**：快速起草、保持风格、提高效率

### 3. 🎯 标题优化

生成 10 个符合公众号爆款特征的标题，按点击率排序。

**适用场景**：标题不够吸引、想要A/B测试、寻找灵感

### 4. 💡 扩写润色

对指定段落进行扩写，增加案例、数据和生动描述。

**适用场景**：内容不够丰满、需要增加细节、提升可读性

### 5. 🔍 提取金句

从文章中提取 10 条具有传播价值的金句。

**适用场景**：制作海报、朋友圈宣传、公众号导语

## 🎓 最佳实践

### 1. 先分析，后生成

不要直接让 AI 写文章，先用"文章解构"分析竞品，再基于分析生成初稿。

### 2. 多轮对话优化

不满意第一次结果？继续追问："标题再短一些"、"语气更专业"、"增加一个案例"

### 3. 融入个人观点

在 Prompt 中加入你的独特见解和案例，让 AI 生成的内容更有个性。

### 4. 人工审核必不可少

AI 生成的内容仅供参考，必须经过人工审核和调整后再发布。

### 5. 遵守平台规范

生成的内容需符合微信公众号运营规范，不得发布违规内容。

## ❓ 常见问题

### Q: 完全免费吗？有使用限制吗？

**A**: 完全免费，无任何使用限制。豆包本身对个人用户免费开放。

### Q: 热点功能显示"加载失败"？

**A**: 确保后端服务已启动：
```bash
cd backend && mvn spring-boot:run
```
然后重新加载Chrome扩展（chrome://extensions/）

### Q: 配图预览界面不显示？

**A**: 检查后端服务状态：
```bash
curl http://localhost:8080/api/ai/health
```
如果失败，重启后端服务。

### Q: 端口8080被占用？

**A**: 杀掉占用进程：
```bash
lsof -ti:8080 | xargs kill -9
```

### Q: 需要API Key吗？

**A**: 不需要。智能配图使用客户端模式，完全免费无需API Key。

### Q: 支持哪些AI客户端？

**A**: 目前支持豆包AI和文心一言，可在配图界面自由切换。

### Q: 生成的内容会重复吗？

**A**: 不会。每次生成都是全新的内容，且有防重复机制。

### Q: 如何保存对话历史？

**A**: 豆包会自动保存所有对话历史，可在豆包官网查看。

更多问题请查看 [完整安装指南](INSTALL_GUIDE.md#常见问题)
